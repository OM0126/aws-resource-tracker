# AWS Resource Tracker

A simple Bash script to track AWS resources using AWS CLI.

#Features

* List S3 Buckets
* List EC2 Instances
* List Lambda Functions
* List IAM Users
* Store output in a file

##  Technologies Used

* AWS CLI
* Bash Scripting
* Linux
* Git & GitHub
* jq

# Project File

* aws_resource_tracker.sh

## How to Run

Clone the repository:

bash
git clone https://github.com/OM0126/aws-resource-tracker.git


Give executable permission:

bash
chmod +x aws_resource_tracker.sh


Run the script:

bash
./aws_resource_tracker.sh
